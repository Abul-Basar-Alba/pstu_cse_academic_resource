# Windows basic commands

## List everything

```
dir
```

## Change directory

```
cd C:\
```

## Create a directory

```
mkdir NewFolder
```

## Remove a directory

```
rmdir NewFolder
```

## Remove a file

```
del file.txt
```

## Copy a file

```
copy source.txt destination.txt
```

## Move a file

```
move a*.txt destination.txt
```

## Rename a file

```
rename oldname.txt newname.txt
```

## Clear the screen

```
cls
```

## Open notepad

```
notepad source.txt
```

## Compile a c code file

```
g++ source.c
```

## Run an executable file

```
source.exe
```

## Display the contents of a file

```
type file.txt
```

## Display directory tree

```
tree
```

## Exit the command prompt

```
exit
```
